| Boundary | Description |
|----------|-------------|
| Assembled multi-page prompt -> ollama.chat | The prompt's user content is untrusted scraped web text; it reaches a local LLM with no tool/action access |
| ollama.chat response -> Pydantic validation | The model's JSON response is treated as untrusted input until `model_validate_json()` succeeds |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
